# 🎀 Smart Productivity System

🧠 A full-stack web application to track daily productivity, habits, and personal performance through structured logging and analytics.

---

## 🚀 Overview

The **Smart Productivity System** helps users monitor their daily routine by logging key metrics like:

- Sleep
- Mood
- Work/Study hours
- Tasks completed
- Distractions

The system provides a structured way to understand productivity patterns and build better habits.

> ⚠️ Machine Learning features (burnout prediction & insights) are planned and currently under development.

---

## ✨ Current Features

### 👤 User Features
- 📝 Daily productivity logging
- ⏱ Track sleep, mood, focus, and work hours
- 📊 Basic productivity tracking
- 📁 Structured data storage via backend
